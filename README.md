# Spotify Analysis

A Python data analysis project exploring trends in top Spotify tracks. Uses the Spotify dataset to uncover patterns in song features like energy, danceability, tempo, and popularity.

## What It Does

- Loads and cleans Spotify track data
- Performs exploratory data analysis (EDA)
- Visualizes relationships between audio features and popularity
- Identifies characteristics of top-charting songs

## Stack

- Python
- Jupyter Notebook
- pandas, matplotlib

## Files

- `Analysis of Top Spotify Tracks.ipynb` — main analysis notebook
